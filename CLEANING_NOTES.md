# Cleaning Notes

The original project files included an assignment cover sheet and submission-style filenames containing personal identifiers. The public/portfolio version has been cleaned by:

- Removing the assignment cover sheet.
- Removing student name, student ID, unit details and due date.
- Excluding screenshots from Oracle APEX that displayed private workspace/schema details.
- Converting the Oracle APEX export into readable standalone SQL scripts.
- Replacing brand-specific product names with generic product labels.
- Creating a concise clean report for portfolio review.
- Excluding original assignment documents from the repository package.

The cleaned repository is intended to demonstrate SQL, dimensional modelling and business intelligence skills without exposing personal academic submission details.
